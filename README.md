## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/yourusername/horbach.dev.git
cd horbach.dev
```

2. Install dependencies:

```bash
yarn install
```

3. Run the development server:

```bash
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
